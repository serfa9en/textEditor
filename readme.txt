На компьютере должна быть установлена java15, иначе файл jar не запустится

1. В терминале Windows переходим в папку где лежит проект (папка Text Editor)
cd "путь к папке"
(Например: cd "C:\*\Project TextEditor\Text Editor")

2. Далее в терминале вводится строка для открытия jar файла
java -jar "путь к файлу jar"
